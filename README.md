# Left-Right LED Flash

### Description:
This is a small program that will linearly flash three LEDs. First they will flash from left to right, and then they will flash from right to left.

### Requirements:
1x MEGA 2560  
1x Breadboard  
6x Wires  
3x 220ohm Resistors  
3x LEDs
